# Browsecomp 推理链模板 (Reasoning Chain Templates)

本文档将 Browsecomp 的 79 个复杂学术问答谜题抽象为 7 个通用的**推理链模板**，每个模板使用团队定义的图谱 Schema 来表示。

> **相关文档**: 参见 [节点选择规则表](./节点选择规则表.md) 了解如何将自然语言约束映射到知识图谱节点和边。

---

## 图谱 Schema 概述

### 节点类型 (Node Types)

| 节点类型 | 描述 |
|----------|------|
| **PaperNode** | 论文本体 |
| **AuthorNode** | 作者/研究者 |
| **InstitutionNode** | 机构（大学、研究机构、医院等） |
| **JournalNode** | 期刊 |
| **ConferenceNode** | 会议 |
| **EntityNode** | 论文相关实体（实验、方法、技术、疾病、化合物等） |

### 边类型 (Edge Types)

| 边类型 | 描述 |
|--------|------|
| **has_author** | 论文→作者（写作关系） |
| **affiliated_with** | 作者→机构（隶属关系） |
| **published_in** | 论文→期刊/会议（发表关系） |
| **cites** | 论文→论文（引用关系） |
| **mentions** | 论文→实体（提及/包含关系） |
| **has_coauthor_with** | 作者→作者（合作关系） |
| **acknowledges** | 论文→人物/实体（致谢关系） |

## 模板 A: 论文-作者-机构链 (Paper-Author-Institution Chain)

### 描述
解决通过论文特征（发表时间、标题词数、引用数、章节结构）查找作者及其所属机构的查询，是出现频率最高的推理模式（约30%）。

### 推理链 (伪代码)

```
起始节点: PaperNode
    [过滤条件]
        publication_year BETWEEN X AND Y
        title_word_count = N
        reference_count = M
        author_count = K

    → has_author → AuthorNode
        [过滤条件]
            author_position = 1st/2nd/3rd
            (OR) birth_date BETWEEN X AND Y
            (OR) has_specific_achievement

        → affiliated_with → InstitutionNode
            [过滤条件]
                founding_date < YEAR
                (OR) is_one_of_oldest_universities
                (OR) establishment_date BETWEEN X AND Y

    → [可选] has_author → AuthorNode (其他作者)
        [过滤条件]
            coauthored_with IN (year1, year2)
```

### 复杂约束组合示例

```
约束组合 1 (来自 Browsecomp #1):
  (PaperNode.publication_year < 2010)
  AND (PaperNode.author_count = 5)
  AND (PaperNode.title_word_count = 8)
  AND (PaperNode.reference_count = 27)
  AND (
    EXISTS AuthorNode IN PaperNode.authors WHERE
    AuthorNode.name = Country.name
  )
  AND (
    COUNT(AuthorNode.affiliated_with) >= 3
    WHERE AuthorNode.affiliated_with = SameInstitution
  )
  AND (
    EXISTS AuthorNode1, AuthorNode2 IN PaperNode.authors WHERE
    EXISTS coauthored_paper(AuthorNode1, AuthorNode2)
    AND coauthored_paper.publication_year = 1994
  )

约束组合 2 (来自 Browsecomp #2):
  (PaperNode.publication_year BETWEEN 2015 AND 2020)
  AND (PaperNode.author_count = 2)
  AND (
    FORALL AuthorNode IN PaperNode.authors:
    AuthorNode.affiliated_with.is_one_of_oldest_universities = TRUE
  )
  AND (
    PaperNode.contains_disclosure_statement = TRUE
    AND PaperNode.disclosure_mentions_initials = ["AF", "AP"]
  )
```

## 模板 B: 人物-学术轨迹链 (Person-Academic-Path Chain)

### 描述
通过人物的学术背景、教育经历、获奖记录、职业时间线来识别特定研究者身份的查询模式（约22%）。

### 推理链 (伪代码)

```
起始节点: AuthorNode (目标人物)
    [过滤条件]
        phd_year = YEAR
        (OR) elected_fellow BETWEEN year1 AND year2
        (OR) award_year = YEAR
        (OR) current_position = POSITION

    → has_author → PaperNode (博士论文/代表作)
        [过滤条件]
            publication_year BETWEEN X AND Y
            title CONTAINS "disabilities"
            (OR) topic = SPECIFIC_TOPIC

    → affiliated_with → InstitutionNode (当前/历史机构)
        [过滤条件]
            founding_date BETWEEN X AND Y
            (OR) institution_type = "liberal_arts_and_sciences"

    → [可选] has_editor_role → JournalNode
        [过滤条件]
            journal_focus = "teaching_practice"
            first_issue_publication BETWEEN X AND Y

    → [可选] has_coauthor_with → AuthorNode (合作者)
        → has_author → PaperNode (合作论文)
            [过滤条件]
                publication_year = award_year
```

### 复杂约束组合示例

```
约束组合 1 (来自 Browsecomp #4):
  (AuthorNode.phd_year = 1983)
  AND (AuthorNode.ams_fellow_year BETWEEN 2005 AND 2020, exclusive)
  AND (
    EXISTS PaperNode, AuthorNodeB, AuthorNodeC WHERE
    PaperNode.authors INCLUDES [AuthorNode, AuthorNodeB, AuthorNodeC]
    AND PaperNode.publication_year BETWEEN 1990 AND 2005
  )
  AND (
    AuthorNodeB.award = "Rollo Davidson Prize"
    AND AuthorNodeB.award_year BETWEEN 1990 AND 2005, exclusive
  )
  AND (
    EXISTS PaperNodeC WHERE
    PaperNodeC.author = AuthorNodeC
    AND PaperNodeC.publication_year BETWEEN 1990 AND 1999
    AND PaperNodeC.title ENDS_WITH "Line"
  )

约束组合 2 (来自 Browsecomp #3):
  (AuthorNode.phd_year = 6th_year)
  AND (AuthorNode.fellowship_topic = "vocal disabilities")
  AND (AuthorNode.fellowship_year BETWEEN 2010 AND 2020)
  AND (
    AuthorNode.current_position.type = "faculty"
    AND AuthorNode.current_affiliation.type = "liberal_arts_and_sciences"
  )
  AND (
    EXISTS JournalNode WHERE
    AuthorNode.editor_of = JournalNode
    AND JournalNode.focus = "improving_teaching_practice"
  )
  AND (
    EXISTS ReportNode WHERE
    ReportNode.published_in = JournalNode
    AND ReportNode.publication_year BETWEEN 2015 AND 2023
    AND ReportNode.volume = "first_issue"
    AND ReportNode.topic = "collaboration_restriction"
  )
```

## 模板 C: 论文引用网络链 (Citation-Network Chain)

### 描述
通过引用关系、参考文献列表、共同引用模式来查找目标论文的查询模式（约15%）。

### 推理链 (伪代码)

```
起始节点: PaperNode (目标论文)
    [过滤条件]
        publication_year BETWEEN X AND Y

    → cites → PaperNode (被引论文)
        [过滤条件]
            publication_year BETWEEN year1 AND year2
            author_count = N
            (OR) title CONTAINS keyword

    → [反向] cited_by → PaperNode (引用该论文的论文)
        [过滤条件]
            reference_number = N
            (OR) cited_position = "first_reference"

    → has_author → AuthorNode
        → has_author → PaperNode (作者的其他论文)
            [过滤条件]
                common_citation EXISTS
```

### 复杂约束组合示例

```
约束组合 1 (来自 Browsecomp #8):
  (PaperNode.topic = "19th_century_African_scholar")
  AND (
    PaperNode.author.country = PaperNode.mentioned_person.country
  )
  AND (
    PaperNode.proposes_resource_sharing_model = TRUE
    AND EXISTS CountryNode WHERE
    CountryNode.area BETWEEN 1000000 AND 1300000  # sq km
    AND CountryNode.has_similar_sharing_pattern = TRUE
  )

约束组合 2 (来自 Browsecomp #3 in 科普+论文):
  (PaperNode.sample_size BETWEEN 150 AND 400)
  AND (PaperNode.author_count = 3)
  AND (
    EXISTS InstitutionNode IN PaperNode.affiliations WHERE
    InstitutionNode.founding_year BETWEEN 1975 AND 2000
  )
  AND (
    EXISTS ReferenceNode IN PaperNode.references WHERE
    ReferenceNode.publication_year BETWEEN 1990 AND 2015
    AND ReferenceNode.topic = "interpersonal_bonding_dynamics"
  )
```

## 模板 D: 多论文合作网络链 (Collaboration-Network Chain)

### 描述
通过多篇论文之间的共同作者关系、作者排列顺序变化来识别特定论文的查询模式（约10%）。

### 推理链 (伪代码)

```
起始节点: PaperNode (论文集合)
    [过滤条件]
        publication_years = [year1, year2, year3, ...]

    → has_author → AuthorNode (共同作者)
        [过滤条件]
            EXISTS_IN_ALL_PAPERS = TRUE
            (OR) author_position_consistent = [1st, 1st, 1st]

    → has_coauthor_with → AuthorNode (其他合作者)
        → [路径合并] → PaperNode (目标论文)
            [过滤条件]
                author_order = [PersonA, PersonB, PersonC]
                is_first_publication = TRUE
```

### 复杂约束组合示例

```
约束组合 1 (来自 Browsecomp #12):
  (
    EXISTS PaperNode1, PaperNode2, PaperNode3 WHERE
    PaperNode1.publication_year BETWEEN 2006 AND 2011
    AND PaperNode2.publication_year BETWEEN 2006 AND 2011
    AND PaperNode3.publication_year BETWEEN 2006 AND 2011
  )
  AND (
    EXISTS AuthorNode1, AuthorNode2 WHERE
    AuthorNode1 IN [PaperNode1.authors, PaperNode2.authors, PaperNode3.authors]
    AND AuthorNode2 IN [PaperNode1.authors, PaperNode2.authors, PaperNode3.authors]
  )
  AND (
    AuthorNode1.affiliation.founding_year BETWEEN 1955 AND 1960
    AND AuthorNode1.current_status = "faculty"
  )
  AND (
    AuthorNode2.affiliation.founding_year BETWEEN 1945 AND 1950
    AND AuthorNode2.previous_employer.founding_year BETWEEN 1840 AND 1850
    AND AuthorNode2.previous_employer.focus = "traffic_operations_safety"
  )
  AND (
    EXISTS AuthorNode3 WHERE
    AuthorNode3 IN [PaperNode1.authors, PaperNode2.authors, PaperNode3.authors]
    AND PaperNode1.is_first_published = TRUE
    AND PaperNode1.author_order = [AuthorNode2, AuthorNode1, AuthorNode3]
  )

约束组合 2 (来自 Browsecomp #7):
  (
    EXISTS Paper2015, Paper2017 WHERE
    Paper2015.publication_year = 2015
    AND Paper2017.publication_year = 2017
    AND Paper2015.authors[0] = Paper2017.authors[0]
    AND Paper2015.authors[2] = Paper2017.authors[2]
  )
  AND (
    EXISTS AuthorNode WHERE
    AuthorNode = Paper2015.authors[2]
    AND AuthorNode.birth_year BETWEEN 1933 AND 1935
    AND AuthorNode.birth_place.name_derived_from = "fire_destroyed_woodland"
  )
  AND (
    EXISTS Paper1998 WHERE
    Paper2015.acknowledges = Paper1998.authors[2]
    AND Paper1998.publication_year = 1998
  )
```

## 模板 E: 活动-参与-成果链 (Event-Participation Chain)

### 描述
通过会议演讲、研讨会参与、机构历史特征等事件来查找相关论文或人物的查询模式（约16%）。

### 推理链 (伪代码)

```
起始节点: EventNode / ConferenceNode
    [过滤条件]
        event_year = YEAR
        event_location = CITY/COUNTRY
        event_type = "conference" / "symposium" / "talk"

    → has_speaker → AuthorNode
        [过滤条件]
            talk_topic = SPECIFIC_TOPIC
            (OR) talk_count = N

    → affiliated_with → InstitutionNode
        [过滤条件]
            founding_date BETWEEN X AND Y
            (OR) undergraduate_admission_2020 = NUMBER

    → has_author → PaperNode (相关论文)
        [过滤条件]
            publication_year BETWEEN X AND Y
            coauthor_publication_count = 1
```

### 复杂约束组合示例

```
约束组合 1 (来自 Browsecomp #13):
  (
    EXISTS ConferenceVisitNode WHERE
    ConferenceVisitNode.year = 2017
    AND ConferenceVisitNode.visited_institution.undergrad_admission_2020 = 3695
    AND ConferenceVisitNode.talk_count = 2
    AND ConferenceVisitNode.talk_topic = "defensive_mechanism_against_invaders"
  )
  AND (
    EXISTS SymposiumNode WHERE
    SymposiumNode.year = 2022
    AND SymposiumNode.speaker = AuthorNode
    AND SymposiumNode.location = ConferenceVisitNode.location
  )
  AND (
    AuthorNode.affiliation.founding_year BETWEEN 1950 AND 1990
  )
  AND (
    EXISTS PaperNode WHERE
    PaperNode.authors INCLUDES [AuthorNode, AuthorNodeB]
    AND AuthorNodeB.publication_count_2020 = 1
    AND PaperNode.publication_year BETWEEN 2010 AND 2015
    AND PaperNode.title CONTAINS punctuation_mark
  )

约束组合 2 (来自 Browsecomp #1 in 询问人物信息):
  (
    EXISTS AuthorNode WHERE
    AuthorNode.supported_proposal = "new_rail_system"
    AND EXISTS CommitteeNode WHERE
    CommitteeNode.investigated = AuthorNode.supported_proposal
    AND CommitteeNode.formation_year BETWEEN 2000 AND 2023
  )
  AND (
    EXISTS InstitutionNode WHERE
    AuthorNode.bachelor_degree_from = InstitutionNode
    AND InstitutionNode.charter_year BETWEEN 1850 AND 1895
    AND EXISTS ActorNode WHERE
    ActorNode.alma_mater = InstitutionNode
    AND ActorNode.starred_in = "comedy_show_1970s"
    AND ActorNode.role = "grumpy_patient"
  )
```

## 模板 F: 技术内容-实体链 (Technical-Content Chain)

### 描述
通过具体技术规格、实验数据、方法学特征、材料属性等来找论文的查询模式（约5%）。

### 推理链 (伪代码)

```
起始节点: PaperNode
    [过滤条件]
        publication_year BETWEEN X AND Y

    → mentions → EntityNode (技术实体)
        [过滤条件]
            entity_type = "coolant" / "alloy" / "compound"
            (OR) technical_parameter_value BETWEEN X AND Y
            (OR) specification = "annealing_temperature > 300°C"

    → [可选] has_author → AuthorNode
        → has_author → PaperNode (其他论文)
            [验证条件]
                cites SAME EntityNode
```

### 复杂约束组合示例

```
约束组合 1 (来自 Browsecomp #1 in 材料):
  (PaperNode.publication_year BETWEEN 2010 AND 2020)
  AND (
    EXISTS EntityNode1 WHERE
    PaperNode.mentions = EntityNode1
    AND EntityNode1.type = "coolant"
    AND EntityNode1.property = "sustainable_alternative"
  )
  AND (
    EXISTS EntityNode2 WHERE
    PaperNode.mentions_application = "machining"
    AND EntityNode2.type = "alloy"
    AND EntityNode2.annealing_temperature BETWEEN 300 AND 800  # °C
    AND EntityNode2.elastic_modulus BETWEEN 10000 AND 18000  # ksi
  )
  AND (
    EXISTS InstitutionNode WHERE
    PaperNode.submitted_to = InstitutionNode
    AND InstitutionNode.student_growth_period = "1850s"
    AND InstitutionNode.student_growth = "rapid"
  )
  AND (
    EXISTS AuthorNode WHERE
    PaperNode.author = AuthorNode
    AND AuthorNode.bsc_year BETWEEN 2005 AND 2013

约束组合 2 (来自 Browsecomp #10 in 科普+论文论文相关信息):
  (
    EXISTS ResearcherNode WHERE
    ResearcherNode.affiliation.undergrad_applicants_2022 = 23819
    AND ResearcherNode.research_topic = "iron_dependent_cell_death"
  )
  AND (
    EXISTS EntityNode WHERE
    EntityNode.type = "cell_death"
    AND EntityNode.dependent_on = "iron"
    AND EntityNode.distinct_from = ["necrosis", "apoptosis"]
  )
  AND (
    ResearcherNode.undergraduate_institution.type = "all_women_university"
    AND ResearcherNode.undergraduate_institution.location = "Asian_country"
  )
  AND (
    COUNT(PaperNode WHERE
      PaperNode.author = ResearcherNode

## 关键约束类型说明

### 时间约束 (Temporal Constraints)
- **时间段**: `BETWEEN x AND y (inclusive/exclusive)`
- **相对时间**: `well into the 20th century`, `late 2000s`
- **事件顺序**: `paper_published AFTER conference_attended`

### 计数约束 (Count Constraints)
- **精确计数**: `author_count = 5`, `reference_count = 27`
- **范围计数**: `sample_size BETWEEN 150 AND 400`
- **存在性计数**: `EXISTS_IN_ALL_PAPERS`, `MORE_THAN 3 authors`

### 属性匹配约束 (Attribute Matching)
- **字符串匹配**: `title ENDS_WITH "problems"`, `name = Country.name`
- **包含关系**: `title CONTAINS "Intervention"`, `topic INCLUDES "vocal disabilities"`
- **精确短语**: `specific_phrase = "infinite faith and love"`

### 逻辑组合约束 (Logical Combinations)
- **AND 组合**: 多个条件必须同时满足
- **OR 组合**: 任一条件满足即可
- **嵌套逻辑**: `(A AND B) OR (C AND D)`
- **存在量词**: `EXISTS`, `FORALL`, `COUNT`

## 使用指南

### 如何应用模板

1. **识别问题类型**: 根据 Browsecomp 谜题的特征，判断属于哪个模板
2. **提取约束条件**: 将自然语言描述转换为 Schema 约束
3. **构建推理链**: 按照模板的伪代码结构构建查询路径
4. **验证约束**: 检查所有约束是否被正确表示

### 示例应用

**Browsecomp #1**:
```
应用模板 A: 论文-作者-机构链

起始: PaperNode (publication_year < 2010, author_count = 5)
